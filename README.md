# 🚚 Company Deliverables System (React & node.js)

A web application for managing products, locations, and delivery journeys, built with React and local APIs.

## 📌 Overview

Work in progress. The app allows users to create and manage delivery journeys by selecting locations and assigning products with quantities to each stop.

### ⚙️ Features
* Create and manage JPG/PNG-products 
* Add and manage locations (with optional coordinates API)
* Build journeys with multiple stops
* Assign products and quantities to each stop
* Fetch and store data via a local backend API
* Interactive UI with modals and dynamic tables

### ▶️ How to Run
* Start the backend server (node server.js)
* Run the React app (npm run dev)
* Enjoy

### 🧠 Concepts Used
* React hooks (useState, useEffect)
* Component-based architecture
* API integration (CRUD operations)
